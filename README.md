# Emotion Detector Project

## 📌 Description
This project is an AI-based Emotion Detection web application developed using IBM Watson NLP. It analyzes the input text and identifies emotions such as joy, sadness, anger, fear, and disgust.

## 🚀 Features
- Detects emotions from user input text
- Displays dominant emotion
- Web interface using Flask
- Error handling for invalid input
- Unit testing included

## 🛠️ Technologies Used
- Python
- Flask
- IBM Watson NLP
- Requests Library
- Pylint (for code analysis)

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/your-username/emotion-detector.git

2. Navigate to project folder:
cd emotion-detector

3. Run the Flask app:
python server.py

4. Open browser:
http://127.0.0.1:5000/

## ✅ Output Example
Input: I am very happy today  
Output: Dominant Emotion → joy
